**Инструкция для запуска проекта**

В первую очередь нужно установить IntellIJ Idea на свой комп. Далее нужно скачать код с этого репозитория (скачиваете архив и потом в верхнем левом угле нажимаете настройки -> open, находите разорхивированный архив и открываете) и поставить его в среду разработки. 

Далее нужно настроить в своем проекте БД, я использовал PostgreSQL, поэтому предлагаю его использовать. Его нужно установить с официального сайта (https://www.postgresql.org/download/). 
В настройках установки кликните на галочку по установке pgAdmin4 (это ПО для визуального управления БД).

Создайте сервер PostgreSQL с портом 5432 (дефолтный порт) и убедитесь, что он работает. В конфигурационном файле application.properties уже настроено подключение к бд, а также настроена видимость
любого изменения в БД засчет работы Spring Boot:

spring.datasource.url=jdbc:postgresql://localhost:5432/postgres
spring.datasource.username=postgres
spring.datasource.password=postgres

ИМЕЙТЕ В ВИДУ, ЧТО ЛОГИН-ПАРОЛЬ-ПОРТ ДОЛЖНЫ СОВПАДАТЬ С ТЕМ, ЧТО НАПИСАНО СВЕРХУ, ИНАЧЕ БУДЕТ ОШИБКА ПРИ ЗАПУСКЕ!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

Теперь нужно один раз запустить проект через класс Main, чтобы БД создалась. 
Далее после одного успешного запуска рекомендуется остановить работу и закинуть в бд какие нибудь данные. Делать это опять же рекомендую через pgAdmin4. Зайдите туда, обновите БД (правой кнопкой мыши по ней, потом refresh). Далее зайти в Schemas -> public -> Tables -> events по "events" правой кнопкой мыши и Query Tool, затем пишите запрос, например:

INSERT INTO events (title, date, description) VALUES ('Матан', '2025-10-01', 'Лекция по матану в аудитории 101');

Запускаете проект, видите результаты
